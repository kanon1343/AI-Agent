# AI エージェント実装

## 説明

https://github.com/openai/openai-agents-python/blob/main/examples/agent_patterns/routing.py
を参考に、簡単な AI エージェントを実装

- リクエストごとにリクエストされた言語で回答

## 使用例

`python3 main.py`
